# Closet-AI 

Closet-AI is an AI-powered fashion recommendation system that helps users choose outfits based on style, color, and occasion. Built with Python, Flask, and machine learning models, this system offers personalized clothing suggestions to users from their own wardrobe or an online catalog.

## Features
- Image-based outfit recommendation
- Style and occasion filters
- Lightweight and user-friendly interface

## Tech Stack
- Python
- Flask
- Selenium (for web scraping)
- Gemini API (for AI suggestions)

## Getting Started
1. Clone the repo
2. Install dependencies using `pip install -r requirements.txt`
3. Run the server with `python server.py`
